# Assignment 2 - Personal Portfolio
A simple, responsive personal portfolio built with HTML, CSS, and JavaScript. The site presents an interactive overview of my background, projects, skills, and a contact form.

## Project description
This portfolio showcases my well-understanding and background in the concept of web development in a clean, minimal layout:
- **Navigation bar** with links to different sections of the page
- **Intro** with interactive buttons to navigate to the cv and GitHub profile
- **About** section with a brief bio
- **Skills** section listing key tools and strengths
- **Projects** grid with example projects mixed between personal and academic work with "...show more" to expand the project descriptions
- **Contact** section with a client-side validated form (no backend)
- **Used technologies:** HTML, CSS, and a JS file.

## Setup instructions
To run the project locally, follow these steps:
1. Clone the repository.
2. Open any code editor (e.g., WebStorm, VS Code) and navigate to the project directory.
3. Right click on `index.html` and select "Open in Browser" to view the portfolio.

## How to use
- Click on the navigation links to jump to different sections of the page.
- In the "Intro" section, click on the buttons to view my CV or GitHub profile.
- In the "Projects" section, click on "...show more" to expand the project descriptions for more details.
- Fill out the contact form and click "Submit" to see the validation in action (note: the form does not actually send data anywhere).
